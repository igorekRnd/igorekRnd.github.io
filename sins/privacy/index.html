Политика конфиденциальности приложения «Грехи»

Приложение «Грехи» предназначено для личной фиксации грехов между исповедями и работает с пользовательскими данными локально на устройстве.

Какие данные использует приложение

Приложение может хранить:

  • названия грехов;
  • факты грехов и связанные заметки;
  • отметки об исповеди;
  • статистические данные, рассчитываемые на основе введённых записей;
  • пользовательские настройки приложения;
  • резервные копии данных, если пользователь сам выполняет экспорт.

Эти данные могут относиться к чувствительной личной информации, поэтому приложение не предполагает их автоматическую передачу.

Как используются данные

Все данные используются только для работы функций приложения:

  • отображения списка текущих грехов;
  • фиксации фактов греха;
  • подготовки списка для Исповеди;
  • расчёта статистики и истории;
  • сохранения пользовательских настроек;
  • создания и восстановления резервных копий.

Передача данных третьим лицам

Приложение не отправляет пользовательские данные на собственный сервер разработчика.

Данные могут передаваться во внешние приложения или сохраняться вне приложения только по явному действию пользователя, например:

  • при сохранении резервной копии в файл;
  • при отправке резервной копии через системное меню «Поделиться»;
  • при импорте данных из выбранного пользователем файла.

Хранение данных

Данные хранятся локально на устройстве пользователя.

Резервные копии создаются только по инициативе пользователя и сохраняются в выбранное им место.

Удаление данных

Пользователь может удалить данные:

  • вручную внутри приложения;
  • удалив резервные копии;
  • удалив приложение и связанные с ним локальные данные.

Дети

Приложение не предназначено специально для детей и не собирает персональные данные детей через внешние сервисы разработчика.

Изменения политики

Политика конфиденциальности может обновляться при изменении функций приложения или требований магазинов приложений.

Контакты

По вопросам, связанным с приложением и обработкой данных, можно использовать email поддержки, указанный в карточке приложения в магазине.
