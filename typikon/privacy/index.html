Политика конфиденциальности приложения «Пост по Типикону»

Приложение «Пост по Типикону» предназначено для просмотра уставного режима питания на выбранную дату и работает без регистрации пользователя.

Какие данные использует приложение

Приложение может хранить локально на устройстве:

  • выбранную пользователем дату;
  • технические настройки интерфейса и состояния экранов;
  • встроенные справочные материалы приложения.

Какие данные приложение не собирает

Приложение не запрашивает учетную запись пользователя и не собирает:

  • имя пользователя;
  • номер телефона;
  • адрес электронной почты;
  • геолокацию;
  • платежные данные;
  • рекламные идентификаторы.

Как используются данные

Локально сохраненные данные используются только для работы функций приложения:

  • открытия выбранной даты;
  • восстановления последнего состояния интерфейса;
  • показа встроенных материалов и ссылок.

Передача данных третьим лицам

Приложение не отправляет пользовательские данные на сервер разработчика и не использует собственные облачные сервисы.

По явному действию пользователя приложение может открыть внешнюю ссылку в браузере, например страницу дня или исходную статью на сайте Азбуки. В этом случае дальнейшая обработка данных происходит по правилам соответствующего сайта или браузера.

Хранение данных

Данные приложения хранятся локально на устройстве пользователя в пределах стандартного хранилища Android.

Удаление данных

Пользователь может удалить данные приложения:

  • очистив данные приложения в настройках Android;
  • удалив приложение с устройства.

Реклама и аналитика

Приложение не показывает рекламу и не использует сторонние системы аналитики.

Дети

Приложение не предназначено специально для детей и не собирает персональные данные детей через внешние сервисы.

Изменения политики

Политика конфиденциальности может обновляться при изменении функций приложения или требований магазинов приложений.

Контакты

По вопросам, связанным с приложением и обработкой данных, можно использовать email поддержки, указанный в карточке приложения в магазине.
