Политика конфиденциальности приложения «Трекер веса»

Приложение «Трекер веса» предназначено для локального ведения записей веса, хранения настроек напоминаний и работы с резервными копиями.

Какие данные использует приложение

Приложение может использовать и хранить:

  • записи веса (дата и значение), введённые пользователем;
  • настройки пола, роста и параметров напоминаний;
  • резервные копии данных, если пользователь создаёт их вручную;
  • временные служебные данные, необходимые для импорта и экспорта.

Как используются данные

Все данные используются только для функций приложения:

  • ведения локального журнала веса;
  • показа динамики веса и диапазона нормальных значений;
  • показа локальных напоминаний;
  • создания и восстановления резервных копий.

Передача данных третьим лицам

Приложение не отправляет пользовательские данные на сервер разработчика и не передаёт их третьим лицам автоматически.

Данные могут быть переданы во внешние приложения только по явному действию пользователя, например:

  • при экспорте резервной копии в файл;
  • при копировании резервной копии в буфер обмена;
  • при открытии JSON-файлов через системные средства Android.

Хранение и удаление данных

Все записи и настройки хранятся локально на устройстве пользователя.

Пользователь может удалить данные:

  • удаляя отдельные записи внутри приложения;
  • очищая все данные через настройки приложения;
  • удалив приложение и связанные с ним локальные данные.

Изменения политики

Политика конфиденциальности может обновляться при изменении функций приложения или требований магазинов приложений.
