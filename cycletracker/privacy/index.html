Политика конфиденциальности приложения «Трекер цикла»

Приложение «Трекер цикла» предназначено для локального ведения записей цикла, ориентировочного прогноза событий и хранения пользовательских настроек.

Какие данные использует приложение

Приложение может использовать и хранить:

  • даты начала и окончания менструации, введённые пользователем;
  • пользовательские настройки длины цикла, длительности менструации и уведомлений;
  • резервные копии данных, если пользователь создаёт их вручную;
  • временные служебные данные, необходимые для импорта, экспорта и показа уведомлений.

Как используются данные

Все данные используются только для работы функций приложения:

  • ведения локального журнала цикла;
  • расчёта ориентировочного прогноза следующей менструации, овуляции и фертильного периода;
  • показа локальных уведомлений по выбранным событиям;
  • создания и восстановления резервных копий.

Передача данных третьим лицам

Приложение не отправляет пользовательские данные на сервер разработчика и не передаёт их третьим лицам автоматически.

Данные могут быть переданы во внешние приложения только по явному действию пользователя, например:

  • при экспорте резервной копии в файл;
  • при отправке резервной копии через системное меню «Поделиться»;
  • при открытии JSON-файлов через системные средства Android.

Хранение данных

Все записи и настройки хранятся локально на устройстве пользователя.

Удаление данных

Пользователь может удалить данные:

  • удаляя отдельные записи внутри приложения;
  • очищая все данные через настройки приложения;
  • удалив приложение и связанные с ним локальные данные.

Медицинские ограничения

Приложение не является медицинским сервисом. Все прогнозы носят ориентировочный характер и не заменяют консультацию врача.

Изменения политики

Политика конфиденциальности может обновляться при изменении функций приложения или требований магазинов приложений.

Контакты

По вопросам, связанным с приложением и обработкой данных, можно использовать email поддержки, указанный в карточке приложения в магазине.
